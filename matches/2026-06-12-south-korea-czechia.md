## Learnings
- The analysis was right about the match and wrong about the slip: the base 3-leg
  builder won. The optional upgrade leg converted a winner into a loser for ~1.5x
  extra payout. #optional-legs
- The "qualifies if combined ≥ X" rule for optional legs is a price test, not a risk
  test — it ignores that a clean-sheet leg adds a failure mode the analysis had
  *already flagged* as Czechia's one strength. New rule: never add an optional leg
  whose kill condition appears in the match's own "what kills it" notes. #clean-sheet-legs
- Don't stack team-scores-zero on match odds + match unders; redundancy bought
  little price for a huge new way to lose. #leg-redundancy
